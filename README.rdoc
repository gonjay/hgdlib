== 说明

觉得学校图书馆的网站对移动端的支持太水了，用相应式针对移动端做了个优化版本
的，搜索的时候速度稍微慢了点，和学校服务器搜索性能有关，也懒得挨个抓书的数
据了，这个样子基本够日常使用了

== 关于微信接口

robots_controller.rb是一个专门处理微信的controller，填上http://yourhostsname.com/robots就
能验证成为微信的开发者了，然后在create方法处理微信服务器post过来的用户消息，这里我在message.rb里
面实现了一些方法，可以实现排队的功能。并写了一个小算法来计算排队所使用的平均时间，这个平均时间是动态
的，会随着前面排队者花费的排队时间计算。
